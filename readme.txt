Create beautiful surveys, polls, and feedback forms for WordPress. Advanced features including NPS, file uploads, heatmaps, and conditional logic.

== Description ==

WPAsk is the most powerful WordPress survey plugin for gathering user feedback, running polls, and measuring customer satisfaction.

**Key Features:**

* **Multiple Question Types** - Rating, NPS, text, email, number, radio, checkbox, dropdown, date, yes/no, and file upload
* **Advanced Targeting** - Show surveys based on page, referrer, time on page, scroll depth, device, and exit intent
* **Conditional Logic** - Skip or show questions based on previous answers
* **Real-time Analytics** - Beautiful charts and aggregated data for each question
* **CSV Export** - Download all responses as CSV files
* **Survey Templates** - Pre-built templates for common use cases
* **Heatmaps** - Visualize where visitors click on your pages
* **Post Ratings** - Add star ratings to posts with shortcodes
* **File Uploads** - Allow users to attach files to their responses
* **Date Range Filtering** - Filter responses by date range

**Use Cases:**

* Customer satisfaction surveys
* Net Promoter Score (NPS) surveys
* Post-purchase feedback
* Content engagement polls
* Website feedback forms
* Lead capture forms
* User research surveys

== Installation ==

1. Upload the `wpask` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to WPAsk → Surveys to create your first survey
4. Use the shortcode `[wpask survey_id="X"]` or the widget to display surveys

== Frequently Asked Questions ==

= How do I create a survey? =

Go to WPAsk → Surveys → Add New Survey. You can choose from pre-built templates or start from scratch.

= Can I customize the survey appearance? =

Yes, you can customize the widget color, position, and confirmation message in the survey settings.

= Does this plugin support file uploads? =

Yes, WPAsk supports file uploads with configurable file type restrictions and size limits.

= Can I export survey responses? =

Yes, you can export all responses as a CSV file from the Results page.

= Does this work with page builders? =

Yes, WPAsk works with all page builders including Elementor, Divi, and Gutenberg using the shortcode.

= Is this plugin GDPR compliant? =

Yes, WPAsk stores all data in your WordPress database and doesn't send data to third-party services.
